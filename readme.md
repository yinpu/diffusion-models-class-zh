# 扩散模型课程

欢迎来到扩散模型课程！

本课程由 Hugging Face 团队的专家们携手打造，旨在：

- 讲解这些模型背后的理论知识。
- 通过使用 `diffusers` 库以及其他 Hugging Face 生态系统中的工具（如 `transformers` 和 `accelerate`），将理论付诸实践。
